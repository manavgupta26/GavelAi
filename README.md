# GavelAI

GavelAI is an AI-powered legal document analysis platform that helps users quickly understand court judgements and legal PDFs. The system extracts important information from uploaded documents and generates structured AI insights for faster review and decision making.

## Features

- Upload and analyze judgement PDFs
- AI-generated summary of legal documents
- Key directions extraction
- Action steps generation
- Risk analysis
- Approval recommendation system
- Clean and simple interface
- Local AI processing using Ollama

## Tech Stack

- Python
- Streamlit
- Ollama
- Llama 3
- PyPDF2 / PDF Processing
- HTML/CSS
